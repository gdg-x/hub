The Hub

Statistics & Metrics

The Hub loves to crunch all the data on GDGs it collects every day. We compute different metrics to track your progress & impact. The interpretation of the data however is up to you.

Data

With over 400 communities world-wide there is a lot of data to be collected and stored each day. This is what the hub does. Running on Google's Cloud Platform we are able to keep track with everything that's happening in the global GDG community.

API

Data should be accessible. This is even more true when it's data that's generated by passionate GDG community members all around the world. It's your data and we can't wait to see what you do with it.
